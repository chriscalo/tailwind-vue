# tailwind-vue

Example of using Tailwind in a Vue.js app. Inspired by Andre Madarang's tutorial
[Tailwind CSS with Vue CLI 3][1].


## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```
